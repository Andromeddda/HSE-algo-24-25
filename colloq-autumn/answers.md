* **–û–ø—Ä. $f(x) = O(g(x))$:** \
$$\exists C>0 \exists x_0 \geq 0: \forall x \geq x_0 : |f(x)| \leq C|g(x)|$$

* **–û–ø—Ä. $f(x) = o(g(x))$:** \

$$\exists 0 < C_1 < C_2 \exists x_0 \geq 0: \forall x \geq x_0 : C_1 |g(x)| \leq |f(x)| \leq C_2 |g(x)|$$

### <ins>9. –ú–∞—Å—Ç–µ—Ä-—Ç–µ–æ—Ä–µ–º–∞ —Å –¥–æ–∫–∞–∑–∞—Ç–µ–ª—å—Å—Ç–≤–æ–º.<ins>

	* –ü—É—Å—Ç—å $T(n)$ - –≤—Ä–µ–º—è —Ä–∞–±–æ—Ç—ã –∞–ª–≥–æ—Ä–∏—Ç–º–∞ –ø—Ä–∏ —Ä–∞–∑–º–µ—Ä–µ –≤—Ö–æ–¥–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö n. \
	–¢–æ–≥–¥–∞ –µ—Å–ª–∏ $T(n) = a T(n / b) + O(n^c)$, —Ç–æ \
	$T(n) = O(n^c)$, –µ—Å–ª–∏ $c > log_a (b)$ \
	$T(n) = O(n^c log(n))$, –µ—Å–ª–∏ $c = log_a (b)$ \

* **–î–æ–∫–∞–∑–∞—Ç–µ–ª—å—Å—Ç–≤–æ:**
	* –ú–∞–∫—Å–∏–º–∞–ª—å–Ω–∞—è –≥–ª—É–±–∏–Ω–∞ —Ä–µ–∫—É—Ä—Å–∏–∏ - $log_b (n)$ \
	–ù–∞ i-—Ç–æ–º —Å–ª–æ–µ –æ–ø–µ—Ä–∞—Ü–∏–π $O(a^i * (\frac{n}{b^i}) ^ c)$ \
	–¢–æ–≥–¥–∞ –Ω–∞ –ø–æ—Å–ª–µ–¥–Ω–µ–º —Å–ª–æ–µ (–≤ –ª–∏—Å—Ç—å—è—Ö) –≤—Å–µ–≥–æ –æ–ø–µ—Ä–∞—Ü–∏–π $O(a^{log_b(n)}) = O(n^{log_b(a)})$ \
	\
	$$T(n) = \sum_{n=1}^{log_b(n)} O(a^i * (\frac{n}{b^i}) ^ c)$$ \
	\
	$$T(n) = O(\sum_{n=1}^{log_b(n)} (a^i * (\frac{n}{b^i}) ^ c))$$ \
	\
	$$T(n) = O(n^c \sum_{n=1}^{log_b(n)} (\frac{a}{b^c}) ^ i)$$ \
	\
	$$\text{Let } q = a/b^c$$ \
	\
	$$T(n) = O(n^c \sum_{n=1}^{log_b(n)} q ^ i)$$ \
	\
	–î–∞–ª—å—à–µ —Ä–∞—Å—Å–º–æ—Ç—Ä–µ—Ç—å –ø–æ —Å–ª—É—á–∞—è–º, $q < 1$, $q = 1$, $q > 1$
